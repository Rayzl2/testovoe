# Тестовое задание Resol
 
**Был разработан WebApi-сервис для обмена данными с БД. Принцип работы:**
_1) Построение базы данных основано на подходе CodeFirst, все изменения БД выполнялись через миграции
2) Работа данных внутри приложения построена через AutoMapper (для полного соответсвия и корректной передачи данных) и MediatR  (для корректного распределения задач внутри проекта)
_

**Папка Carts.Identity - отдельный проект с сервером**
**Папка Carts.Application - проект с построением основных команд и функций**
**Папка Carts.Persistence - проект отвечающий за инициализацию и подключением к БД**
**Папка Carts.WebApi - проект отвечающий за взаимодействие со всем проектом**

**Библиотеки которые я использовал:**
_1) Psqgl
2) EntityFramework
3) Automapper
4) Swagger
5) MedaitR
6) IdentityServer4
7) FluentValidation_
